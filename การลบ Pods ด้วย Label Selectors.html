การลบ Pods ด้วย Label Selectors

วิธีที่ทรงพลังในการจัดการทรัพยากรจำนวนมาก

หลักการทำงาน

นอกจากการลบ Pod ทีละตัวตามชื่อแล้ว เรายังสามารถใช้ Label Selector เพื่อลบ Pod หลายๆ ตัวพร้อมกันได้ในคำสั่งเดียว ซึ่งสะดวกและทรงพลังมาก หลักการคือ เราจะใช้ Label ที่เรากำหนดไว้ให้กับ Pod ต่างๆ เป็นเงื่อนไขในการเลือก ถ้า Pod ไหนมี Label ตรงตามที่เราเลือก มันก็จะถูกลบทั้งหมด

ตัวอย่างเช่น เราอาจจะลบ Pod ที่เป็นเวอร์ชันทดสอบ (canary) ทั้งหมดได้ง่ายๆ โดยใช้ selector อย่าง rel=canary โดยไม่กระทบกับ Pod ที่เป็นเวอร์ชัน stable เลย

ภาพประกอบ

ตัวอย่างคำสั่ง

1. ลบ Pods จาก Label ที่เจาะจง

# สั่งลบ pod ทั้งหมดที่มี label "creation_method=manual"
# -l คือ flag สำหรับระบุ label selector
$ kubectl delete po -l creation_method=manual

2. ตัวอย่างการลบ Pods ที่เป็น Canary Release

# สั่งลบ pod ทั้งหมดที่มี label "rel=canary"
$ kubectl delete po -l rel=canary
